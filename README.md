# 👥 HR Analytics Dashboard

## 📌 Project Overview

The **HR Analytics Dashboard** is an interactive Power BI solution designed to help Human Resources teams analyze workforce data, monitor employee performance, and identify trends related to attrition, demographics, and organizational performance. The dashboard provides actionable insights that support strategic HR decision-making.

---

## 🎯 Project Objective

The primary objective of this project is to:

* Monitor employee workforce metrics.
* Analyze employee attrition and retention.
* Evaluate department-wise employee distribution.
* Track salary and demographic trends.
* Support HR decisions with data-driven insights.

---

## 🛠️ Tools & Technologies

* Microsoft Power BI
* Microsoft Excel
* Power Query
* DAX (Data Analysis Expressions)

---

## 📂 Dataset

The dataset includes employee-related information such as:

* Employee ID
* Department
* Job Role
* Gender
* Age
* Salary
* Experience
* Education
* Attrition Status
* Joining Date

---

## 📈 Key Performance Indicators (KPIs)

* Total Employees
* Active Employees
* Attrition Count
* Attrition Rate (%)
* Average Salary
* Average Age

---

## 📊 Dashboard Features

* Executive KPI Cards
* Employee Distribution by Department
* Attrition Analysis
* Gender Distribution
* Age Group Analysis
* Salary Analysis
* Interactive Filters and Slicers

---

## 💡 Business Insights

* Identifies departments with higher attrition.
* Provides workforce demographic analysis.
* Monitors employee distribution across departments.
* Supports HR planning and workforce management.
* Helps improve employee retention strategies.
